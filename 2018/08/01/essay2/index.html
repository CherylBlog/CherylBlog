发表于 2018-08-01

搭建步骤

在 GitHub 上创建仓库

登录到 GitHub 账号，如果没有 GitHub 账号，可以注册。然后点击 GitHub 的 New repository 创建一个新的仓库，它的名字应该是你的 GitHub 的 username.githun.io，这种是固定写法。如果你的用户名是 abc，那么你的仓库名成就应该是 abc.github.io。
